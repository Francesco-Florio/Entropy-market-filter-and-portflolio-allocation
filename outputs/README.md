# 📊 Generated Outputs

This folder contains the reproducible tables and figures generated by `unified_entropy_portfolio.ipynb`. The committed files correspond to the latest successful execution and are intentionally tracked by Git.

Return to the [main README](../README.md) or open the [figure guide](figures/README.md).

## 📈 Main Backtest Files

| File | Rows | Description |
| --- | ---: | --- |
| `daily_portfolio_returns.csv` | 5,382 | Daily regime, entropy thresholds, weight cap, strategy return, equal-weight return, S&P 500 return, and solver status |
| `wealth_curves.csv` | 5,382 | Cumulative wealth indices for the three compared strategies |
| `drawdown_curves.csv` | 5,382 | Daily drawdown paths for the three compared strategies |
| `weights_long.csv` | 53,820 | Ticker-level daily weights for the entropy MVO and equal-weight portfolios |
| `diagnostics.csv` | 5,382 | Solver messages, weight sums, extrema, active-name count, and constraint diagnostics |

## 📋 Summary Tables

| File | Description |
| --- | --- |
| `performance_metrics.csv` | Annualized return, volatility, Sharpe ratio, maximum drawdown, and final wealth by strategy |
| `phase_summary.csv` | Number and share of Bullish, Neutral, and Bearish days with average entropy and mean strategy return |
| `transition_counts.csv` | Observed counts of transitions between entropy regimes |
| `transition_probabilities_4dp.csv` | Row-normalized regime transition probabilities rounded to four decimals |
| `crisis_analysis.csv` | Performance statistics by strategy across four predefined stress windows |

## 🖼️ Figures

The `figures/` folder contains nine PNG files covering the toy examples, cumulative performance, drawdowns, stress periods, regime diagnostics, transition probabilities, and portfolio weights.

See [figures/README.md](figures/README.md) for an explanation and preview of every image.

## 🔎 Interpretation Notes

- `EntropyMVO` identifies the entropy-regime allocation strategy.
- `EqualWeightTop10` is the dynamic equal-weight benchmark over the same active universe.
- `SP500` is the broad-market benchmark.
- `SolverSuccess` records whether SLSQP returned a successful solution; Bearish equal-weight allocations intentionally skip optimization.
- Returns and metrics are historical backtest outputs and should not be interpreted as forecasts.

## ♻️ Regeneration

Running the notebook recreates this folder. Existing generated figures are cleared before the main plotting stage, while the two toy-example figures are preserved and regenerated by their dedicated cells.
